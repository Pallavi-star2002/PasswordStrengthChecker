## 🔍 Password Validation Criteria

The application checks:

- Password Length
- Lowercase Letters (a-z)
- Uppercase Letters (A-Z)
- Numbers (0-9)
- Special Characters (!@#$%^&*)

### Strength Levels

| Score | Status |
|---------|---------|
| 0 | Very Weak |
| 1 | Weak |
| 2 | Medium |
| 3 | Strong |
| 4 | Secure |
